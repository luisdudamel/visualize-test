## Commands

```shell
    # Installation command
    npm i or npm/yarn i

    # Running command
    npm run dev

    # Build command
    npm run build

    # Lint
    npm run lint
```

```shell
    # Testing command
    npm run test

    # Testing watch command
    npm run test:watch

    # Testing coverage command
    npm run test:coverage
```
